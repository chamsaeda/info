🌐 IP 주소와 도메인 이름

인터넷 주소의 비밀을 알아볼까요?

🎯 학습 목표

컴퓨터가 인터넷에서 서로를 찾아가는 방법을 이해하고, IP 주소와 도메인 이름의 관계를 배워봅시다!

🏠

IP 주소란?

IP 주소는 인터넷에 연결된 컴퓨터의 고유한 주소예요. 마치 우리 집의 도로명 주소처럼 말이죠!

192.168.0.1

숫자 4개를 점(.)으로 연결한 형태
각 숫자는 0~255 사이의 값

💡 예를 들면?
집 주소: 서울시 강남구 테헤란로 123
IP 주소: 192.168.0.1

📝

도메인 이름이란?

도메인 이름은 외우기 어려운 IP 주소 대신 사용하는 쉬운 이름이에요!

www.naver.com

사람이 기억하기 쉬운 문자 형태
알파벳, 숫자, 하이픈(-) 사용 가능

💡 왜 필요할까?
223.130.195.95 ❌ 외우기 어려워요!
www.naver.com ✅ 쉽게 기억해요!

🔄 DNS: 도메인을 IP로 변환하기

DNS (Domain Name System)는 도메인 이름을 IP 주소로 바꿔주는 시스템이에요. 마치 전화번호부처럼 "이름 → 번호"를 찾아주는 역할을 합니다!

✅ 변환 완료!

🏠 IP 주소

💡 이제 컴퓨터는 이 IP 주소를 이용해 해당 웹사이트에 접속할 수 있어요!

🤔 왜 IP 주소로는 접속이 안 될까?

🎭 상황

브라우저에 223.130.195.95를 입력했어요. → 하지만 접속이 안 돼요! 왜 그럴까요?

🏢 이유 1: 한 건물에 여러 회사가 있어요

하나의 서버에 여러 웹사이트가 함께 있을 수 있어요. 마치 한 건물에 여러 회사가 있는 것처럼요!

🏢 빌딩 (IP: 192.168.1.100)
• 1층: www.naver.com
• 2층: www.daum.net
• 3층: www.kakao.com

❌ 건물 주소만 알고 가면 어느 층으로 가야 할지 모르겠죠?

✅ 해결책: 도메인 이름을 사용하면 정확한 층을 알려주는 거예요!

🔒 이유 2: 보안 때문이에요 (HTTPS)

요즘 웹사이트는 HTTPS 보안을 사용해요.

🔐 보안 인증서

HTTPS는 "이 사이트가 진짜인가요?"를 확인하는 인증서를 사용해요.

인증서: "이것은 www.naver.com용입니다"

❌ IP 주소로 접속하면 인증서가 거부해요!

🌍 이유 3: 전 세계에 흩어져 있어요 (CDN)

큰 웹사이트는 전 세계 여러 곳에 서버를 두고 있어요.

🗺️ 유튜브의 서버들
• 🇰🇷 한국: 142.250.207.46
• 🇺🇸 미국: 142.250.217.46
• 🇯🇵 일본: 142.250.196.46

💡 도메인으로 접속하면 자동으로 가까운 서버로 연결돼요!

📌 정리

IP 주소는 건물 주소만 알려주지만, 도메인 이름은 정확한 목적지와 신원까지 알려줘요!

🎮 참새다! 이해도 체크 퀴즈

Q1. 다음 중 올바른 IP 주소 형식은?

정답! IP 주소의 각 숫자는 0~255 사이의 값이어야 해요.

Q2. DNS의 역할은 무엇일까요?

정답! DNS는 사람이 기억하기 쉬운 도메인 이름을 컴퓨터가 이해하는 IP 주소로 바꿔줘요.

Q3. 왜 도메인 이름을 사용할까요?

정답! 192.168.1.1 같은 숫자보다 www.naver.com 같은 이름이 훨씬 기억하기 쉽죠!
