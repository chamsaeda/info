💻 컴퓨터와 데이터의 세계

컴퓨터가 정보를 처리하는 원리를 알아볼까요?

🎯 학습 목표

이 학습 자료를 통해 컴퓨터가 데이터를 표현하는 원리(비트·바이트·경우의 수)부터 인터넷 통신의 기초(IP 주소·네트워크 장비·무선 통신), 그리고 데이터 압축 기술(무손실·손실·JPEG)까지 폭넓게 이해할 수 있습니다. 각 주제를 직접 체험하며 컴퓨터와 인터넷의 작동 원리를 배워봅시다!

비트(bit)와 바이트(byte)

컴퓨터 데이터의 기본 단위 이해하기
1

비트(bit) → 0 또는 1, 가장 작은 단위

2

바이트(byte) → 8개의 비트 = 1바이트

3

데이터 용량 → KB, MB, GB, TB

💡 핵심 개념: 비트, 바이트, 데이터 용량 단위
